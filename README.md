# webrtc-janus

WebRTC 미디어 서버인 Janus Gateway 에 대해서 간략하게 정리한 내용입니다. Janus Gateway Plugin 테스트 위한 로컬 환경(Vagrant + VirtualBox)을 구성 했으며, Go 프로그램으로 미디어 파일을 실행하여 Janus Gateway Plugin과 연동를 하였습니다.
